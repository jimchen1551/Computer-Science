# Maximum likelihood
$$\mathbf{M}(q)=\arg\max_{l\in levels(t)}P(q[1],\dots,q[m]|t=l)$$
- aka **ML** prediction
- returning the target level with the highest conditional probability given the state of the descriptive features in the query
