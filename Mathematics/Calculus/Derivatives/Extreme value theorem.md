# Extreme value theorem
$$\text{if }f(x)\text{ is continuous on }[a, b],$$
$$\text{then there exist numbers }c\text{ and }d\text{ s.t. }$$
1. $$a\leq c,d\leq b$$
2. $$f(c)\text{ is the absolute maximum in }[a,b]$$
3. $$f(d)\text{ is the absolute minimum in }[a,b]$$
