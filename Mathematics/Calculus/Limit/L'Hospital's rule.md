# L'Hospital's rule
$$\text{if }\lim_{x\rightarrow a}\frac{f(x)}{g(x)}=\frac{0}{0}\text{ or }\lim_{x\rightarrow a}\frac{f(x)}{g(x)}=\frac{\pm\infty}{\pm\infty},$$
$$\text{then }\lim_{x\rightarrow a}\frac{f(x)}{g(x)}=\lim_{x\rightarrow a}\frac{f'(x)}{g'(x)}$$
