# Steepest Descent
![[Pasted image 20230222165322.png|500]]
$$\Delta x_k=-\alpha_k\nabla f(x_k)$$
$$\alpha_k=\arg\min_{\alpha}f(x_k+\alpha \Delta x_k)$$
- updating the learning rate for each iteration, compared with [[Gradient descent]]
